# PG Manager New

Owner-first PG/Hostel Management SaaS.

## Projects

- `backend`: Spring Boot 3, Java 21, Gradle, MySQL, Flyway, JWT security.
- `owner_app`: Flutter owner mobile application.
- `docs`: Architecture, API, database, and implementation notes.

## Phase 1

Phase 1 focuses on owner operations:

- Owner registration and login.
- Organization/property/floor/room/bed setup.
- Tenant onboarding.
- Bed assignment, transfer, checkout, and history.
- Rent and payment tracking.
- Owner dashboard.

Build and run commands are intentionally not executed yet because local software installation will be completed later.
